# Sistema de Gestión de Producción - ACME

## Descripción
Este proyecto es una aplicación web para la gestión de producción de la empresa "ACME". Permite administrar la información del sistema mediante una interfaz intuitiva con autenticación de usuarios y diferentes módulos de gestión.

## Funcionalidades

- Inicio de sesión de usuarios.
- Registro de nuevos usuarios.
- Consulta de información.
- Creación de nuevos registros.
- Edición de registros existentes.
- Eliminación de registros.
- Visualización de los datos en tarjetas organizadas.

## Estructura del Proyecto

```
├── index.html          # Redirige automáticamente al inicio de sesión
├── login/              # Módulo de autenticación
├── usuarios/           # Gestión de usuarios
├── procesos/           # Gestión de procesos
├── productos/          # Gestión de productos
└── css/                # Archivos de estilos
```

## Funcionamiento

1. Al abrir el proyecto se redirige automáticamente a la pantalla de "Login".
2. El usuario puede iniciar sesión o registrarse.
3. Una vez autenticado, puede acceder a los diferentes módulos del sistema.
4. Cada módulo permite realizar operaciones "CRUD":
   - Crear
   - Consultar
   - Actualizar
   - Eliminar registros

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Características

- Diseño responsive.
- Interfaz moderna con tema oscuro.
- Formularios organizados y fáciles de usar.
- Navegación sencilla entre módulos.
